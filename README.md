# VantaShala-Automation

## Project setup
```
npm install
```

### To open cypress for first time
```
npx cypress open
``` 

### Add Typescript files to tsconfig.json to compile
```
example: "files": [ "core.ts","sys.ts"]
               ( or )
         "include": ["src/**/*"]
```


### Run a compile based on a backwards look through the fs for tsconfig.json
```
tsc 
```

### Run your tests
```
npm run test:local
```
